# ml-models

`ml-models` is a Python package aimed at providing a collection of machine learning models for various tasks.

## Testing

All tests in this project are conducted using the Iris dataset from scikit-learn.

To run the tests for this project, you'll need to have pytest installed. If you haven't installed it yet, you can do so via pip:

```bash
pip install pytest
```

Once pytest is installed, you can run the tests using the following command:


```bash
pytest -s
```

The `-s` flag is used to display print statements during test execution in order to see the metrics.
